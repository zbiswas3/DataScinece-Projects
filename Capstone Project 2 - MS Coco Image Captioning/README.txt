Problem Definition

Build a machine learning model that will take an image as input and generate a descriptive caption of the image. 

----------------------------------------------------------------
Dataset Used

Due to the smaller scope of the project and keeping all other constraints before us, we decided to use COCO 2017 dataset to train the model. The dataset contains 5000 images with 20,000 captions (4 captions for each).
